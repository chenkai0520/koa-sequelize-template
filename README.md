# koa-template
一个带有用户注册登陆功能的koa项目模版

# scripts
+ `npm run start`, 生产环境启动项目
+ `npm run dev`,`开发环境下启动项目,内容改变会自动重启服务
+ `npm run test`, 允许jest单元测试
+ `npm run lint`, eslint检查并格式化代码
+ `npm run initdb`，初始化项目需要的数据库环境
